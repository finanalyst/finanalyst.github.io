Using RakuAST to render RakuDoc to HTML

The next upgrade of the Rakudo Raku compiler uses RakuAST to parse programs. This has a major implication for RakuDoc sources, which can be manipulated after being parsed, but before being compiled. RakuDoc itself has been upgraded.

Overview

The RakuAST::RakuDoc::Render distribution contains several renderers, including two for HTML. All the HTML files in this repository were rendered using RakuDoc::To::HTML-Extra.
